Capture by Interface

How technical systems become mechanisms of institutional control and degradation

Framework

Capture by Interface describes how control of technical interfaces enables institutional takeover and system degradation. Unlike traditional forms of capture that rely on direct control, interface capture operates by making institutions dependent on technical systems they don't control.

Key Insight

Technical interfaces don't just mediate control - they become active mechanisms of capture, enabling system-wide control through administrative bottlenecks.

Key Mechanisms

Interface Dependency

Making basic operations dependent on technical interfaces controlled by external entities

Administrative Bottlenecks

Creating technical chokepoints that enable system-wide control

Capacity Stripping

Degrading institutional ability to operate independently

System Degradation

Converting direct capabilities into platform-mediated services

Historical Examples

Treasury Payment Systems (2025)

Control of federal payment interfaces enabling system-wide administrative capture

Gig Nursing Platforms (2024)

Healthcare delivery made dependent on platform interfaces
